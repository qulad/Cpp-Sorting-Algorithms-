# C++ Sorting Algorithms
Sorting algorithms in C++

To use it just ```#include "filename"``` 
- Bubble Sort:
  - bubbleSort_SingleThread(T arr[], unsigned int size)
  - bubbleSort_Multithread(T arr[], unsigned int size)

T could be int, float or double.
